## v0.6.11

### Highlights

This release adds **Docker image publishing to GHCR**, fixes **PyPI CI publishing**, and reduces the **public faucet daily budget** to 1,000 Token/day.

### Features

- **Docker image on GHCR**: The `release` workflow now builds and pushes multi-platform (linux/amd64 + linux/arm64) Docker images to `ghcr.io/claw-network/clawnet` on every `v*` tag.
- **Dual-registry npm publishing**: All npm packages are published to both [npmjs.org](https://www.npmjs.com/org/claw-network) and [GitHub Packages](https://github.com/orgs/claw-network/packages) via CI.
- **PyPI publishing via CI**: Python SDK (`clawnet-sdk`) is automatically published to [PyPI](https://pypi.org/project/clawnet-sdk/) on tag push.

### Fixes

- Fixed PyPI CI publish by adding `HATCH_INDEX_USER=__token__` environment variable (was prompting for username interactively).

### Changes

- Reduced public faucet daily budget default from 10,000 → **1,000 Token/day** (`CLAW_FAUCET_DAILY_BUDGET`).
- Updated `skills/publish-packages.md` with full dual-registry + CI workflow documentation.
